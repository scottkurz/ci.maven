Open Liberty

The Open Liberty project is an open community supporting Jakarta EE and MicroProfile.

The Open Liberty server provides a runtime that starts fast and is made for running cloud-native microservices.

Odo

The odo CLI tool is an iterative and straightforward way for developers to write, build, and deploy applications on Kubernetes and OpenShift. The odo tool abstracts away complex Kubernetes and OpenShift concepts for the developer.

One of Odo's core concepts is the concept of a devfile. A devfile is portable file that describes a development environment, which allows you to deploy your applications with ease.
